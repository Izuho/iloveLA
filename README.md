## 文法の制限
- 行列の中身は整数
- 変数は小文字一文字
- 変数定義は where 以下で行う
- 変数は実数の範囲でのみ定義する
- 'ℝ'の代わりに'R'
- '×'の代わりに'*'
- '∈'の代わりに':'
- その他、適用できるLA文法はとても狭い

## 例
'''
t^3+s^5
where
t:R
s:R
'''
